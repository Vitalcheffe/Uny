# Contributing to UNY

## Development Workflow

1. **Fork the repo**
2. **Create a feature branch**
   ```bash
   git checkout -b feature/amazing-feature
   ```

3. **Make your changes**
4. **Test thoroughly**
   ```bash
   npm test
   npm run lint
   npm run build
   ```

5. **Commit with conventional commits**
   ```bash
   git commit -m "feat: add amazing feature"
   ```

6. **Push and create PR**
